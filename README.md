# 🌻 小农场 -缄默之秋— SillyTavern 助手脚本

在末世酒馆中经营一片属于自己的小农场，种植作物、应对随机事件、收获存入庇护所。

---

## 快速开始

1. 在酒馆助手中启用本脚本，屏幕上会出现一个 🌻 气泡图标。
2. 点击气泡打开农场面板，选择一个庇护所即可开始种植。
3. 点击停用或在酒馆助手中关闭脚本，数据自动保存，下次打开恢复。

---

## 基本操作

### 种植

- 点击空地（虚线边框的格子）打开作物选择器。
- 选择一种作物种下，作物会经历 **种子 → 发芽 → 生长 → 成熟 → 可收获** 五个阶段。
- 农场共有 **12 块田地**（3 列 × 4 行）。

### 收获

- 作物到达「可收获」阶段后，点击该田地即可收获，产出自动存入当前庇护所。
- 也可以点击 **一键收获** 按钮批量收获所有成熟作物。

### 查看作物详情

- 点击正在生长的作物，弹出详情面板，可查看水分、肥力、健康值和剩余时间。
- 点击「关闭」按钮或点击弹窗外的暗色区域均可关闭详情。

## 道具系统

### 道具一览

| 道具 | 图标 | 效果 | 对应事件 |
|------|------|------|----------|
| 水壶 | 💧 | 浇水 +35，解除干旱 | 干旱 |
| 除草剂 | ✂️ | 健康 +20，清除杂草 | 杂草 |
| 除虫剂 | 🧴 | 健康 +25，消灭害虫 | 虫害 |
| 肥料 | 🧪 | 肥力 +35，恢复肥力 | 土地贫瘠 |

### 使用方式

- **电脑端**：将道具拖拽到目标田地上释放，**每次只消耗 1 个**。
- **手机端**：先点击道具选中（高亮），再点击目标田地使用，同样每次 1 个。
- 道具不会一次性用完，请放心使用。

### 道具恢复

- 每种道具最多持有 **10 个**。
- 每 **2 分钟** 自动恢复 1 个（所有种类同时恢复），无需手动操作。
- 初始各持有 2 个。

## 随机事件

种植过程中可能触发以下事件，触发后作物 **停止生长**，需要使用对应道具解除：

| 事件 | 图标 | 影响 | 解除方式 |
|------|------|------|----------|
| 干旱 | 🏜️ | 水分持续下降，生长停滞 | 拖拽 💧 水壶到田地 |
| 杂草 | 🌿 | 健康缓慢下降 | 拖拽 ✂️ 除草剂到田地 |
| 虫害 | 🐛 | 健康快速下降，可能减产 | 拖拽 🧴 除虫剂到田地 |
| 土地贫瘠 | 🪨 | 肥力持续下降，产量降低 | 拖拽 🧪 肥料到田地 |
| 小偷 | 🦝 | 有人来偷庄稼！ | 点击田地驱赶 |

> 事件发生时气泡图标会变为橙色闪烁提醒。如果放任不管，作物健康归零会 **枯死**。

## 庇护所

- 农场与酒馆中的庇护所系统联动，收获的作物会存入当前选中的庇护所。
- 每个庇护所拥有 **独立的农场数据**（田地、道具、收获记录互不影响）。
- 通过面板顶部的下拉框切换庇护所，切换时自动保存当前数据并加载目标庇护所。
- 点击 🔄 按钮可手动刷新庇护所列表。

## 界面操作

### 气泡图标

- **点击**：打开 / 关闭农场面板。
- **拖拽**：自由移动到屏幕任意位置，松手后位置自动保存。

### 农场面板

- **顶部标题栏**：可拖拽移动整个面板。
- **底部导航栏**（停用按钮所在行）：同样可拖拽移动面板。
- **右下角三角**：拖拽可调整面板大小。
- 面板每次打开时自动 **居中显示**，适配手机和电脑。

### 按钮说明

| 按钮 | 位置 | 功能 |
|------|------|------|
| 🌑 / ☀️ | 顶部 | 切换暗色 / 亮色主题 |
| 🔄 | 顶部 | 刷新庇护所列表 |
| 清除农田 | 底部 | 清空所有田地（不影响收获记录） |
| 🛑 停用 | 底部 | 停用农场，移除 UI，数据保留 |

## 数据与多端同步

### 自动保存

以下操作会自动保存数据，无需手动操作：

- 种植、收获、使用道具
- 切换庇护所
- 事件触发与解除
- 点击「停用」按钮
- 脚本被酒馆助手关闭或页面卸载时

### 多端同步

农场数据通过 **MVU（SillyTavern 服务端）** 实时同步：

- 每次保存操作同时写入本地 IndexedDB 和 MVU。
- 脚本启动时自动从 MVU 拉取最新数据覆盖本地。
- **手机和电脑访问同一个酒馆实例时，种植进度、收获记录、道具数量完全同步。**
- 切换设备后打开农场即可看到另一端的最新状态。

### 停用与恢复

- 点击「🛑 停用」会保存所有数据后移除农场 UI。
- 下次在酒馆助手中重新启用脚本，所有数据自动恢复：正在种植的作物、收获记录、道具余量等。

## 移动端适配

- 面板大小根据屏幕尺寸自动调整（小屏手机自动缩小、切换 2 列布局）。
- 气泡图标和面板均支持触摸拖拽。
- 道具使用采用「点选 + 点击田地」方式，无需长按拖拽。
- 所有弹窗（作物详情、收获确认、事件提示）均适配触屏操作。
